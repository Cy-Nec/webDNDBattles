# WebFight — D&D Combat Tracker

Приложение для отслеживания боя в D&D.

## Запуск

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск сервера (для сохранения данных)
```bash
npm run server
```

### 3. Запуск приложения (в другом терминале)
```bash
npm run dev
```

## Функционал

- Добавление участников (Игроки, NPC)
- Настройка ХП и инициативы для NPC
- Выбор отношения NPC (Друг/Враг)
- Режим боя с сортировкой по инициативе
- Управление ХП в режиме боя (−/+)
- Сохранение данных в JSON файл

## Структура данных

Данные хранятся в `data/participants.json`:
```json
{
  "participants": [
    {
      "id": 1234567890,
      "type": "player",
      "name": "Арагорн",
      "hp": 45,
      "initiative": 15,
      "friend": true
    }
  ]
}
```
